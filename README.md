# competition-doc-standards

创新创业大赛文档制作规范 Skill for Claude Cowork

基于第八至第十二届互联网+创新创业大赛真实文件的深度分析，覆盖110个DOCX、50个XLSX、71个PDF和6个PPTX文件，提炼出的可直接复用的制作标准。

## 适用场景

制作互联网+（中国国际大学生创新大赛）相关的 Word文档、Excel表格、PDF文件、PPT演示文稿时自动加载此规范。

## 内容覆盖

- **Word文档**：页面设置、字体字号（公文/学术两种风格）、表格规范、图文排版
- **Excel表格**：三种标准模板（信息登记表/汇总表/评分表）、列宽行高、合并策略、数据验证
- **PDF文件**：页面尺寸、字体嵌入、排版技巧、导出设置
- **PPT演示**：版式结构、配色方案、布局技巧、动画规范
- **通用规范**：文件命名、文件夹组织、排版检查清单

## 安装

将 SKILL.md 文件放入 Claude Cowork 的 skills 目录即可。
